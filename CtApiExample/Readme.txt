CTAPI methods are described in detail in the Vijeo Citect Technical Reference that is shipped with PowerSCADA.  
An excerpt of this document has been included with this project.

To run this example, you must have PowerSCADA running on the local machine.  Place the exe file in the Citect bin folder.

If you would like to run your application on a remote machine, refer to the first page of the attached PDF for instructions.

This example does not factor in redundancy.
